=== MainWP AAM Extension ===
Plugin Name: MainWP AAM Extension
Plugin URI: https://aamportal.com
Description: AAM extension to connect all sites to MainWP Dashboard
Version: 1.0.0
Author: Vasyltech
Author URI: https://vasyltech.com
Documentation URI: https://aamportal.com/integrations/mainwp

== Installation ==

1. Please install plugin "MainWP Dashboard" and active it before install MainWP AAM Extension plugin (get the MainWP Dashboard plugin from url:https://mainwp.com/)
2. Upload the `mainwp-aam-extension` folder to the `/wp-content/plugins/` directory
3. Activate the MainWP AAM Extension plugin through the 'Plugins' menu in WordPress

== Screenshots ==
1. Enable or Disable extension on the "Extensions" page in the dashboard
