== Installation ==

1. Please install plugin "MainWP Dashboard" and active it before install MainWP AAM Extension plugin (get the MainWP Dashboard plugin from url:https://mainwp.com/)
2. Upload the `mainwp-aam-extension` folder to the `/wp-content/plugins/` directory
3. Activate the MainWP AAM Extension plugin through the 'Plugins' menu in WordPress
