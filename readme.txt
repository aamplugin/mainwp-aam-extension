AAM extension to connect all sites to MainWP Dashboard

== Description ==

This extension integrates Advanced Access Manager (AAM) with MainWP, enabling seamless synchronization of AAM security scan results with your MainWP dashboard.

With this initial release, you can effortlessly monitor security concerns across your managed network of sites, ensuring better visibility and control from a single interface.

**We welcome your feedback and feature requests!**

Your suggestions help shape future updates, and we’re committed to implementing them promptly. If you have any ideas or concerns, feel free to submit out [contact form](https://aamportal.com/contact-us#other), and we’ll prioritize them for upcoming releases.

== Installation ==

1. Please install plugin "MainWP Dashboard" and active it before install AAM Extension for MainWP plugin (get the MainWP Dashboard plugin from url:https://mainwp.com/)
2. Upload the `aam-extension-mainwp` folder to the `/wp-content/plugins/` directory
3. Activate the AAM Extension for MainWP plugin through the 'Plugins' menu in WordPress

== Screenshots ==
